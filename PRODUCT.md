# FloatDeck - macOS 悬浮幻灯片演示器

## 一句话描述

一个始终置顶的 macOS 悬浮窗应用，以小卡片形式显示 PDF/网页缩略图，点击或快捷键丝滑放大到全屏演示，再缩回小卡片。

---

## 核心概念

FloatDeck 有两种窗口状态：

| 状态 | 说明 |
|------|------|
| **卡片模式 (Card)** | 小尺寸悬浮窗，始终置顶，显示当前页面缩略图。默认比例取自 PDF 首页宽高比（如 A4 = 1:1.414）。可拖拽移动、缩放。 |
| **演示模式 (Present)** | 全屏演示尺寸，居中展示当前内容，适合投屏/讲解。 |

两种模式之间通过动画丝滑切换。

---

## 窗口控制

**不使用** macOS 默认红绿灯（traffic light）窗口按钮。替换为自定义的两个方形悬浮图标按钮：

| 按钮 | 位置 | 功能 |
|------|------|------|
| **缩小** (Minimize) | 左上角第一个 | 从演示模式缩回卡片模式 |
| **关闭** (Close) | 左上角第二个 | 关闭当前文档/退出应用 |

- 按钮风格：圆角方形，半透明背景，hover 高亮
- 卡片模式下仅显示关闭按钮（已是最小状态）
- 演示模式下显示缩小 + 关闭两个按钮

---

## 支持的内容类型

### 1. PDF 文件
- 拖拽 PDF 文件到窗口加载
- 通过菜单 / 快捷键 `Cmd+O` 打开
- 左右翻页浏览不同页面
- 每页支持上下滚动（适配超长页面）
- 首页宽高比决定卡片模式默认尺寸比例

### 2. 网页 (URL)
- 通过地址栏或 `Cmd+L` 输入 URL 加载
- 支持上下滚动浏览网页内容
- 左右翻页无效（网页为单页滚动模型）

## 交互设计

### 模式切换

| 操作 | 效果 |
|------|------|
| 卡片模式下 **单击窗口** | 放大到演示模式 |
| 卡片模式下按 **`Space`** | 放大到演示模式 |
| 演示模式下按 **`Esc`** | 缩小到卡片模式 |
| 演示模式下 **点击缩小按钮** | 缩小到卡片模式 |
| 演示模式下按 **`Space`** | 缩小到卡片模式 |

切换动画：窗口从卡片位置/尺寸 **平滑缩放+移动** 到屏幕中央全屏尺寸（spring 动画，~0.4s）。

### 翻页（PDF 模式）

| 操作 | 效果 |
|------|------|
| **`←`** 或 **`↑`** | 上一页 |
| **`→`** 或 **`↓`** | 下一页 |
| 两指左右滑动 | 翻页 |

### 滚动

- PDF：鼠标滚轮 / 触控板上下滚动当前页面内容
- 网页：鼠标滚轮 / 触控板上下滚动

### 窗口操作（卡片模式）

| 操作 | 效果 |
|------|------|
| 拖拽标题栏区域 | 移动窗口位置 |
| 拖拽窗口边缘/角落 | 按比例缩放窗口大小 |

## 快捷键汇总

| 快捷键 | 功能 |
|--------|------|
| `Space` | 切换卡片/演示模式 |
| `Esc` | 演示模式 → 卡片模式 |
| `←` / `→` | 翻页（PDF） |
| `Cmd+O` | 打开 PDF 文件 |
| `Cmd+L` | 输入 URL 加载网页 |
| `Cmd+W` | 关闭当前文档 |
| `Cmd+Q` | 退出应用 |

## 窗口尺寸

### 卡片模式
- **默认宽度**：280pt
- **默认高度**：根据 PDF 首页比例自动计算（如 A4 → 280 × 396pt）
- 网页默认比例：16:10（280 × 175pt）
- 用户可自由拖拽边角等比缩放，范围 160pt ~ 480pt 宽
- 记住上次关闭时的位置和大小

### 演示模式
- 居中显示，尺寸为屏幕可用区域的 90%（保留边距）
- 保持内容原始宽高比

---

## 视觉设计

- **窗口样式**：无标题栏，圆角（12pt），带阴影
- **背景**：半透明毛玻璃效果（NSVisualEffectView）
- **自定义按钮**：24×24pt 圆角方形，SF Symbols 图标
  - 缩小按钮：`arrow.down.right.and.arrow.up.left`
  - 关闭按钮：`xmark`
- **页码指示器**（PDF）：底部居中，小圆点或 `3 / 12` 文字
- **内容区域**：卡片模式下内容等比缩放填充；演示模式下原始尺寸渲染

## 技术方案

| 模块 | 技术选型 |
|------|---------|
| 框架 | SwiftUI + AppKit（窗口控制需要 NSWindow） |
| PDF 渲染 | PDFKit（`PDFView`） |
| 网页渲染 | WebKit（`WKWebView`） |
| 窗口管理 | `NSWindow`：`.styleMask` 去掉标题栏，设置 `.level = .floating` 置顶 |
| 动画 | SwiftUI `withAnimation(.spring())` + `NSWindow.setFrame(_:display:animate:)` |
| 拖拽加载 | `onDrop` / `NSWindow` 拖拽代理 |
| 持久化 | `UserDefaults` 存储窗口位置、大小、最近文件 |

### 关键实现要点

1. **去掉红绿灯按钮**：`window.styleMask = [.borderless, .resizable]`，`window.titlebarAppearsTransparent = true`，隐藏标准按钮
2. **始终置顶**：`window.level = .floating`
3. **拖拽移动**：borderless 窗口需手动实现 `mouseDragged` 移动
4. **等比缩放**：监听窗口 resize，保持宽高比约束
5. **丝滑动画**：记录卡片模式的 frame，切换时 animate 到目标 frame

## 文件结构（规划）

```
FloatDeck/
├── FloatDeckApp.swift          # App 入口，窗口配置
├── WindowManager.swift         # 窗口状态管理（卡片/演示切换、frame 计算）
├── ContentView.swift           # 主视图容器
├── CardView.swift              # 卡片模式视图
├── PresentView.swift           # 演示模式视图
├── PDFContentView.swift        # PDF 渲染视图（PDFKit 封装）
├── WebContentView.swift        # 网页渲染视图（WebKit 封装）
├── CustomTitleBar.swift        # 自定义标题栏（缩小/关闭按钮）
├── PageIndicator.swift         # 页码指示器
├── DragDropHandler.swift       # 文件拖入处理
└── Assets.xcassets/            # 图标资源
```
